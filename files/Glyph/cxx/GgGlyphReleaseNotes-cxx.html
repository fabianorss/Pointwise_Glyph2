Topic: pw::Rule Changes for 2025.2.1

  • pw::Rule.setCriterion (CHANGED) - Disabled all non-quality related metrics from being used with rules.  Topic: pw::Script Changes for 2025.2.1

pw::BlockStructured Changes for 2025.1

  • pw::BlockStructured.NormalExplicitSmoothingScalingExponent (ADDED) - This specifies/retrieves the exponent used in the scaling function for explicit smoothing.  The exponent controls how aggressively smoothing is applied as the extrusion moves away from the wall.  Values closer to zero ramp up the smoothing more quickly.  A value of zero reverts to Fidelity Pointwise’s legacy scaling function.
  • pw::BlockStructured.NormalVolumeSmoothingIterations (ADDED) - This specifies/retrieves the number of local point-Jacobi volume smoothing iterations to perform at each layer.  A value of zero disables local point-Jacobi smoothing and reverts to Fidelity Pointwise’s legacy volume smoothing algorithm.
  • pw::BlockStructured.NormalVolumeSmoothingBlend (ADDED) - This specifies/retrieves the coefficient controlling the degree of additional smoothing to apply with the local point-Jacobi volume smoothing algorithm.  This attribute is only used when point-Jacobi volume smoothing is enabled.
